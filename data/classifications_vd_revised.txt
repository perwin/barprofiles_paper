# Classifications of (scrambled-order) bar-major-axis profile types for S4G galaxies
# This means: S4G barred spirals which are either low-inc (i < 30) *or* moderate-inc
# and good PA (deltaPA_bar,dp <= 60)
# by Victor P. Debattista
# 
#    Profile = bar-major-axis profile, from r ~ 0 to end of bar
#    Suggested types:
#       Exp = single-exponential
#       Exp(N) = single-exponential with nuclear excess
#       Flat-top = roughly constant central SB, exponential falloff
#       Flat-top(N)
#       Two-slope = outer steep exponential, inner shallow exponential
#       BP = classic EE85 "flat" profile associated with B/P bulges
#
#    Outer_slope = approximate slope of profile outside the bar, realtive
#    to the profile inside (closest to bar end if multiple slopes in bar)
#    Suggested types:
#       steeper, same, shallower
#
# NOTE: This file incorporates revised classifications made by VD, sent to PE
# on 9 May 2019: these were his reclassifications (sometimes unchanged) of those
# 27 galaxies where his classifications differed from PE's, possibly because he
# had forgotten about the option of "Two-Slope" ("2S") classifications
# Classifications which were changed are indicated by "[CHANGED]".
#
